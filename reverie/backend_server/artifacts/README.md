# Artifacts Directory

Generated content from agent creative actions.

| Folder | Contents |
|--------|----------|
| `writings/` | Full text of books, letters, essays, poems, journal entries |
| `paintings/` | Image generation prompts describing artwork |
| `songs/` | Song lyrics and musical compositions |
| `meals/` | Detailed meal descriptions with ingredients and preparation |
| `inventions/` | Descriptions of inventions and crafted items |

`artifact_log.jsonl` — Append-only log of all generated artifacts.
